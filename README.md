# CONTROLE DE ENTREGAS - DELÍCIAS DA NENA

Aplicativo Android desenvolvido em Kotlin que permite ao usuário marcar as entregas que foram concluídas e visualizar o que ainda está pendente de envio, Mantendo assim o controle das entregas que ainda estão pendente de envio e do que já foi entregue. 

O aplicativo foi desenvolvido para o trabalho de extensão da estacio, disciplina #Programação para Dispositivos Móveis, com o intuito de atender a necessidade da empreendedora.

# Funcionalidades: 

1 - Marcar ou desmarcar as entregas que foram concluídas.

2 - Salvamento do status das entregas utilizando SharedPreferences.

# Como utilizar  

1 - Clone o repositório

2 - Abra o projeto no Android Studio.

3 - Compile e execute o aplicativo em um emulador ou dispositivo físico.
